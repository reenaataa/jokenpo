# Jokenpô — Pedra, Papel e Tesoura
![preview](./imgs/preview.png)
✊ Descrição
---------
Jogo simples de Jokenpô (Pedra, Papel e Tesoura) para navegador, feito acompanhando material disponibilizado pela [Dev Samurai](https://class.devsamurai.com.br/). 

Consideravelmente fácil de fazer pra quem conhece JS, mas para mim, que não sei neca de pitibiribas, foi um pouco puxado. Levei exatas 5 horas e 10 minutos.

✋ Principais funcionalidades
-------------------------
- Jogada do usuário contra CPU
- Regras clássicas: Pedra > Tesoura, Tesoura > Papel, Papel > Pedra
- Resultado (vitória, derrota ou empate)
- Contagem simples de placar

✌️ Como jogar
----------
1. Escolher uma opção: Pedra, Papel ou Tesoura.  
2. O computador escolhe aleatoriamente.  
3. O jogo compara escolhas e exibe o resultado e o placar.
